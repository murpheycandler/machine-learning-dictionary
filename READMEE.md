# 🤖 ML Dictionary

A personal, browser-based dictionary for Machine Learning terms and concepts. All data is stored locally in your browser's localStorage.

## Features

- ✏️ **Add & Edit Terms**: Easily add new ML terms with definitions and edit existing ones
- 🔍 **Search**: Quick search through terms and definitions with highlighting
- 📊 **Multiple Sort Options**:
  - Alphabetical (A-Z, Z-A)
  - Date Added (Newest/Oldest)
  - Date Modified (Most Recent)
- 💾 **Persistent Storage**: All data is saved automatically in your browser
- 📱 **Responsive Design**: Works on desktop and mobile devices
- 🎨 **Clean Interface**: Modern, intuitive UI for easy navigation

## How to Use

### Online Version
Visit the live version at: `https://[your-username].github.io/ml-dictionary`

### Local Usage
1. Clone this repository
2. Open `index.html` in your web browser
3. Start adding your ML terms!

### Data Storage
All dictionary entries are stored in your browser's localStorage. This means:
- Your data stays on your device
